# python-beginner-projects
Collection of beginner python projects (calculator, number guessing game, and a maths quiz)

This repository contains beginner-level Python projects built while learning programming fundamentals.

Projects

1.  Calculator

- Performs basic arithmetic operations
- Uses conditionals and user input

2.  Number Guessing Game

- User guesses a number
- Uses loops and conditions

3.  Maths Quiz

- Multiple questions with score tracking
- Uses loops, conditions, and logic building

Skills Used

- Python basics
- if / elif / else
- loops (range)
- variables
- user input handling

Note:

These projects were built as part of my early Python learning journey.

▶️ How to Run

1. Install Python
2. Download the files
3. Run in terminal:

python calculator.py
python guessing_game.py
python math_quiz.py
